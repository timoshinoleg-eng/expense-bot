# Expense Bot Secure v3 - Final

Корпоративный бот для авансовых отчетов с whitelist в Google Sheets.

## Структура Google Sheets

### Лист 1 - Расходы
Имя | Фамилия | Дата время | Сумма | Статья расходов | Объект | File ID чека

### Лист 2 - Сотрудники
ID | Имя | Фамилия | Статус | Роль

Пример:
123456789 | Иван | Петров | Активен | Админ
834868627 | Мария | Иванова | Активен | Сотрудник

## Команды

Для всех:
- /start - приветствие
- /add - добавить трату
- /view - просмотр чека
- /getid - получить Telegram ID

Только админ:
- /add_employee - добавить сотрудника
- /remove_employee - заблокировать сотрудника
- /stats - статистика
- /users - список пользователей

## Установка

1. Создай Google Sheets с двумя листами
2. Второй лист назови "Сотрудники"
3. Добавь заголовки: ID | Имя | Фамилия | Статус | Роль
4. Добавь себя: твой_id | Имя | Фамилия | Активен | Админ
5. Положи service_account.json в папку проекта
6. Заполни .env
7. python main.py
